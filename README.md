# lyric_stream

A new Flutter project.

## Getting Started

Nama Proyek: lyric_stream (Lyriya)
Teknologi Utama: Flutter, Firebase Authentication, & Firestore.

lyriya ini ada hasil project akhir yang saya kerjakan untuk memenuhi tugas sebagai member GDGoC. Dalam proses pengembangannya, proyek ini diselesaikan melalui studi mandiri dengan merujuk pada berbagai sumber edukasi video, serta memanfaatkan bantuan teknologi Kecerdasan Buatan (AI) sebagai mitra diskusi dalam penyempurnaan logika pemrograman dan struktur kode.

Fitur Utama
* Autentikasi: Implementasi Login & Register menggunakan Firebase Authentication.
* Manajemen Data (CRUD): Kemampuan untuk menampilkan (Read), menambah, serta memanipulasi data melalui integrasi database.
* Struktur Proyek: Pemisahan antara UI dan logika yang bersih sesuai standar pengembangan aplikasi mobile.
